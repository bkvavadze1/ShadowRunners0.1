A fast, atmospheric infinite runner built in Unity.  
Dodge obstacles, slide under hazards, and sprint through a collapsing medieval city as you race toward the last bell.

> **Playable build:** https://danteitch.itch.io/shadow-runners

---

## Features
- Dynamic 3-lane movement (jump / slide / lane switch)
- Randomized obstacle generation
- Multiple difficulty modes
- Particle-based dash / impact effects
- Local high score tracking
- Clean UI & atmospheric lighting

## Tech Stack
- **Engine:** Unity 6 (6000.0)
- **Language:** C#
- **Packages:** Cinemachine, Input System
- **Version Control:** Git (code-first; large art/audio excluded)
